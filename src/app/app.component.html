Slider with Step

  • HTML
  • .ts
  • styles.css
         <range-slider [min]="min1" [max]="max1" [step]="step" (onRangeChange)="rangeChanged($event)" [(ngModel)]="stepRange" [highlightClass]="'stephighlightClass'" [barClass]="'stepBarClass'" [sliderClass]="'stepSliderClass'">        </range-slider>
                            
                                min1=-10;
                                max1=50;
                                step=5;
                                stepRange=[0,40];

                                width:90%;
                                height:10px;
                                background:lightgray;
                                border-radius:5px;
                                }
                            
                            .stepSliderClass

                                width:20px;

                                background:lightgray;
                                border-radius:50%;    
                                border:1px solid royalblue;

                                }

                                background:royalblue;
                                }

Two way binding

Min:
Max:
  • HTML
  • .ts
 <range-slider [min]="min" [max]="max" [toolTips]="[false,false]" [(ngModel)]="twoWayRange">        </range-slider>
                    
                          min=-20;
                          max=60;
                          twoWayRange=[10,30];
                          //oninput change you need to do
                          changed(){
                            this.twoWayRange=[...this.twoWayRange];
                            }

Using with Template Driven Forms

  • HTML
  • .ts
 <form (ngSubmit)="onSubmit()" #dummyForm="ngForm" >
                        <div class="form-group">
                        <label for="name">Range</label>
                        <range-slider [min]="min" [max]="max"[step]="4" [(ngModel)]="dumyForm.range" (onRangeChange)="rangeChanged($event)" name="range" #rangef="ngModel" required >
                        </range-slider>
                        </div>
                        <button type="submit" class="btn btn-success" [disabled]="!dummyForm.valid">Submit</button>
                        </form>
                
                        dumyForm = {
                            range: [9,38]
                            }
